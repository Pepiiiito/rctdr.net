# Web portfolio RCTDR (multi-pagina)

Lloc estatic llest per pujar al domini (sense build tools).

## Pagines

- `index.html`: portada.
- `projecte.html`: context i valor del projecte.
- `proces.html`: cronologia tecnica.
- `videos.html`: galeria de videos.
- `recursos.html`: STL i documentacio.
- `contacte.html`: formulari i FAQ.

## Arxius comuns

- `styles.css`: estil visual i responsive.
- `app.js`: interaccions (menu, modal, FAQ, formulari) i contingut editable.

## Personalitzacio rapida

Edita `PROJECT` a `app.js`:

- `ownerName`
- `contactEmail`
- `videos[]` (titol, descripcio, embed)
- `resources[]` (titol, descripcio, enllac)

## Deploy

Puja tots els `.html` + `styles.css` + `app.js` al hosting de `rctdr.net`.
